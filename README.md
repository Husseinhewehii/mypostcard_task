## Installation


>> Download Project 

* https:
``` bash
git clone https://github.com/Husseinhewehii/mypostcard_task.git
```
* ssh:
``` bash
git clone git@github.com:Husseinhewehii/mypostcard_task.git
```

>> Move To Project Directory

``` bash
    cd mypostcard_task
```


>> Install Composer Packages

``` bash
composer install
```
